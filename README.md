## ğŸš€ How to Use

Follow these simple steps to start watching the channels from this playlist.

### Step-by-Step Guide

1.  **Copy the Playlist URL**:
    Right-click on the raw M3U file link from this repository and copy the URL. The URL should look something like this:
    ```
    https://raw.githubusercontent.com/projectdevphil/iptv-playlist/refs/heads/new-path/cignal/index.m3u
    ```

2.  **Open Your IPTV Player**:
    Launch your preferred IPTV application from the [Compatible Apps](#-compatible-apps) list below.

3.  **Add the Playlist**:
    Look for an option to "Add Playlist," "Add M3U URL," or a similar button (often a **+** icon).

4.  **Paste the URL**:
    In the input field, paste the URL you copied in Step 1. Give the playlist a name (e.g., "Project Dev Playlist") and save it.

5.  **Load Channels & EPG**:
    The application will now load the channels and the Electronic Program Guide (EPG) data from the URL specified in the M3U file. Enjoy the streams!

## ğŸ“± Compatible Apps

This playlist is designed to work with any modern IPTV player that supports M3U playlists and MPEG-DASH streams. For the best experience with Clearkey DRM, we recommend using players that properly handle KODIPROP tags.

| Platform      | Recommended Apps                                |
|---------------|-------------------------------------------------|
| **Android TV / Google TV** | TiviMate, IPTV Smarters Pro, Kodi       |
| **Mobile (Android/iOS)**   | IPTV Smarters, GSE Smart IPTV           |
| **Windows / macOS / Linux**| VLC Media Player, Kodi, IPTV Smarters Pro |
| **Smart TV (Tizen/WebOS)** | Smart IPTV, SS IPTV                     |

> **Note:** For players like Kodi, you need to have the **InputStream Adaptive** add-on installed and enabled for MPEG-DASH streams to work correctly.
